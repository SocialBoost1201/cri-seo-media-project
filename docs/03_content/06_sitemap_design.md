## 2. サイト階層

```
TOP
↓
カテゴリ
↓
ピラー記事
↓
クラスター記事
```

この構造によりSEO評価の集中とカテゴリ評価の強化を実現する。

## 8. URL設計方針

| 方針 | 理由 |
|------|------|
| 短く意味のある構造 | 検索エンジンが理解しやすい |
| 英語スラッグを使用 | 正規化・URL共有時の可読性 |
| 階層は2段階まで | クロール効率の最大化 |

**例：**
- `/it/agent`（良）
- `/category/subcategory/article-name`（避ける）
