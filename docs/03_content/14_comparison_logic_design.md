転職領域はYMYL（Your Money or Your Life）ジャンルに該当するため、
透明性・公平性・再現性のある比較ロジックを設計する。

## 1. 比較ロジック設計の目的

1. ユーザーの意思決定支援
2. 客観的評価の提示
3. ランキングの透明性確保

## 4. 評価スコア設計

| スコア | 意味 |
|--------|------|
| 5 | 非常に優れている |
| 4 | 優れている |
| 3 | 標準 |
| 2 | やや弱い |
| 1 | 弱い |

---

## 5. 重み付け設計

| 指標 | 重み |
|------|------|
| 求人数 | 25% |
| サポート品質 | 25% |
| アドバイザー品質 | 20% |
| 対応業界 | 15% |
| ユーザー評価 | 15% |
| **合計** | **100%** |

---

## 6. 総合評価算出方法

```
総合スコア = Σ（各評価スコア × 重み）
```

**例：**
```
求人数(4) × 0.25 = 1.00
サポート品質(5) × 0.25 = 1.25
アドバイザー品質(4) × 0.20 = 0.80
対応業界(3) × 0.15 = 0.45
ユーザー評価(4) × 0.15 = 0.60
─────────────────────────
総合スコア = 4.10
```
