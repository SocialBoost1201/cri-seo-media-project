# 転職SEOメディア構築プロジェクト
コンテンツ設計書

> 作成者ペルソナ：SEOコンテンツストラテジスト / デジタルメディア編集長 / 検索意図分析アーキテクト

本メディアは「検索流入獲得」「転職意思決定支援」「求人送客」の3つを目的とし、
SEO記事・比較記事・意思決定記事の3種類を中心に設計する。

---

## 1. コンテンツ戦略の基本方針

| 方針 | 説明 |
|------|------|
| 検索意図ベース | ユーザーが検索する理由を起点に記事を作成する |
| 意思決定支援 | 単なる情報ではなく転職判断を支援する内容にする |
| 比較中心 | 転職サービスの比較を中心とする |
| 内部リンク強化 | 記事同士を内部リンクで接続する |

## 4. 記事構造テンプレート

すべての記事は以下の構造で作成する。

```
タイトル
↓ リード文
↓ 結論
↓ 比較表
↓ ランキング
↓ 詳細解説
↓ メリット / デメリット
↓ FAQ
↓ 関連記事
↓ CTA
↓ まとめ
```
