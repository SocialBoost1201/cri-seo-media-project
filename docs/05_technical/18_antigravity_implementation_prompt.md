---

## プロジェクト概要

転職SEOメディアのデモサイトを構築する。

| 項目 | 内容 |
|------|------|
| 目的 | SEOメディア構造の可視化 / 転職情報提供 / 転職サービス比較 / 求人送客導線の提示 |
| 既存求人メディア | Webist |

```
検索流入 → 情報提供 → 比較 → 意思決定 → 求人 → Webist
```
