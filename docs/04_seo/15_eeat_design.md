# 転職SEOメディア構築プロジェクト
E-E-A-T設計書

> 作成者ペルソナ：Google Search Quality Strategist / YMYL SEOコンサルタント / コンテンツ信頼性アーキテクト

転職領域はYMYL（Your Money or Your Life）ジャンルに該当するため、
通常のSEOメディアよりも強い信頼性設計が必要である。
本設計書では **E-E-A-T（Experience / Expertise / Authoritativeness / Trustworthiness）** の4要素を定義する。

---

## 1. E-E-A-T設計の目的

1. 検索エンジンの信頼評価を高める
2. ユーザー信頼を向上させる
3. YMYLコンテンツ品質基準を満たす
4. SEOランキングの安定性を高める

---

## 2. Experience（経験）

実体験や実務経験を示す要素。

| 施策 |
|------|
| 転職体験コンテンツ |
| 転職成功事例の掲載 |
| キャリアアドバイザーコメント |
| 実際の転職プロセス解説 |

## 4. Authoritativeness（権威性）

| 施策 | 内容 |
|------|------|
| 運営会社情報 | 企業概要・実績の掲載 |
