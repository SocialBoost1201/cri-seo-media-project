## 3. 検索意図分類

転職検索は主に4つの検索意図に分類される。

| 検索意図 | キーワード例 | 本メディアの対応 |
|---------|------------|----------------|
| 情報収集 | 転職とは / 転職するべきか / 転職 不安 | **獲得対象** |
| 比較 | 転職エージェント 比較 / IT転職エージェント | **獲得対象** |
| 意思決定 | おすすめ転職エージェント / IT転職 エージェント おすすめ | 補完的に対応 |
| 行動 | 求人 / 転職サイト登録 | Webistへ送客 |

本メディアは「**情報収集**」「**比較**」の2つの検索意図を中心に獲得する。

## 5. コンテンツクラスター戦略

```
ピラー記事（例：転職エージェントおすすめ）
├── クラスター記事：転職エージェント 比較
├── クラスター記事：転職エージェント 大手
└── クラスター記事：転職エージェント 失敗
```

この構造によりカテゴリ単位のSEO評価を強化する。
