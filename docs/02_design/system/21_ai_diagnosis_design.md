# 転職SEOメディア デモサイト
AI診断機能 設計書

> 作成者ペルソナ：AIプロダクト開発ディレクター / UXコンバージョン最適化アーキテクト / 診断コンテンツ設計スペシャリスト

本機能は本番AI連携ではなく、提案・打ち合わせ用の **高品質デモ機能** として実装する。

---

## 1. 機能の位置付け

```
検索 → 記事 → AI診断 → おすすめカテゴリ → おすすめサービス → 求人 → Webist
```

SEOメディアの一部として自然に組み込む。

---

## 2. 機能概要

| 項目 | 内容 |
|------|------|
| 機能名 | AIキャリア診断 |
| URL | `/ai-career-diagnosis` |
| 目的 | ユーザーの転職志向・状況・希望条件から最適な転職カテゴリとサービスを提示 |
| 実装方式 | ローカル状態管理 / ダミーロジック / 将来AI API差替可能 |

---

## 3. 診断体験フロー

| ステップ | 内容 |
|---------|------|
| 1 | 導入画面（説明 + 開始ボタン） |
| 2 | 質問ステップ（1問ずつ表示、6問） |
| 3 | 診断中演出（0.8〜1.5秒のローディング） |
| 4 | 結果画面（タイプ + サマリー + おすすめ） |
| 5 | CTA表示 |

---

## 4. 質問設計（6問）

| No | 質問 | 選択肢 |
|----|------|--------|
| Q1 | 転職を考えている一番大きな理由は？ | 年収を上げたい / キャリアアップしたい / 働き方を改善したい / 未経験領域に挑戦したい |
| Q2 | 現在の年齢帯に最も近いものは？ | 20代前半 / 20代後半 / 30代前半 / 30代後半以上 |
| Q3 | 現在の仕事への満足度は？ | かなり不満 / やや不満 / どちらともいえない / 大きな不満はない |
| Q4 | 転職で重視したいことは？ | 年収 / 成長環境 / 安定性 / ワークライフバランス |
| Q5 | あなたに近いタイプは？ | 論理的に考える / 人と関わるのが得意 / 新しいことを学ぶのが好き / 慎重に安定して進めたい |
| Q6 | 今の状況に最も近いものは？ | 未経験職種も検討したい / 今の職種で条件アップしたい / 早めに転職したい / まずは情報収集から始めたい |

---

## 5. 診断結果パターン（5タイプ）

| タイプ | 対応カテゴリ | 推奨CTA |
|-------|------------|---------|
| IT転職向き | `/it` | IT転職向けサービスを見る |
| 第二新卒転職向き | `/second-career` | 第二新卒向け記事を見る |
| 営業転職向き | `/sales` | 営業転職カテゴリを見る |
| ハイクラス転職向き | `/executive` | ハイクラスエージェントを見る |
| 転職エージェント活用向き | `/agent` | おすすめ転職エージェントを見る |

---

## 6. 判定ロジック

各回答にカテゴリごとのスコアを付与し、最も高いカテゴリを結果とする。

```
総合スコア = Σ（各質問の選択肢スコア）
結果 = max(IT, 第二新卒, 営業, ハイクラス, エージェント活用)
```

`lib/diagnosis.ts` にロジックを分離し、将来AI APIへ差替可能にする。
