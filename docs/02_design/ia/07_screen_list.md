# 転職SEOメディア構築プロジェクト
画面一覧書

> 作成者ペルソナ：UXプロダクトディレクター / Webアプリケーション情報設計責任者 / SEOメディアUIアーキテクト

---

## 1. 画面一覧設計の目的

1. 実装対象画面を明確化する
2. 各画面の目的と役割を整理する
3. UI設計・情報設計・ワイヤーフレーム設計の基礎情報とする
4. デモサイトとして見せるべき画面優先度を明確にする

---

## 2. 実装方針

本デモサイトはSEOメディア構造・送客導線・UI体験の理解を目的とするため、
全機能を網羅するのではなく **戦略説明に必要な主要画面** に絞って実装する。
