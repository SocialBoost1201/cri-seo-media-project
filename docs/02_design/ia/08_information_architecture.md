# 転職SEOメディア構築プロジェクト
情報設計書（Information Architecture）

> 作成者ペルソナ：Information Architecture Director / SEOプロダクトストラテジスト / UXコンテンツアーキテクト

---

## 1. 情報設計の基本方針

| 原則 | 説明 |
|------|------|
| 検索意図ファースト | ユーザーが検索した目的を最短距離で満たす構造にする |
| 結論ファースト | 「結論 → 比較 → 詳細」の順で情報を配置する |
| 意思決定支援 | 単なる情報ではなく、比較・判断材料を提供する |
| 回遊設計 | 関連記事・人気記事・カテゴリ導線を設置し回遊率を高める |

---

## 2. ページ共通情報構造

すべてのページは以下の基本構造で設計する。

```
ヘッダー
↓
パンくず
↓
メインコンテンツ
↓
関連記事
↓
CTA
↓
フッター
```
