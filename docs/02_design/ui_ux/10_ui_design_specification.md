# 転職SEOメディア構築プロジェクト
UIデザイン設計書

> 作成者ペルソナ：UIデザインディレクター / プロダクトデザインアーキテクト / デザインシステムストラテジスト

本サイトは単なるブログ型SEOサイトではなく、上場企業の提案デモサイトとして
**信頼性・知性・先進性・プロダクト品質** を同時に表現するUI設計とする。

---

## 1. UIデザインコンセプト

| 項目 | 内容 |
|------|------|
| コンセプト | **Intelligent Career Platform** — 知的で信頼性の高いキャリア意思決定支援メディア |
| デザイン印象 | 信頼性 / プロフェッショナル / 未来志向 / 知的 |
| 参考UI | Notion / Stripe / Linear / Vercel |
| UIスタイル | モダンプロダクトUI + SaaS UI + ナレッジメディアUI の統合 |

---

## 2. デザイン原則

| 原則 | 説明 |
|------|------|
| 読みやすさ | 情報密度を高めつつ視認性を維持する |
| 意思決定支援 | 比較・ランキング・評価が直感的に理解できるUI |
| カード型UI | 記事・サービス・比較をカードで整理する |
| 余白設計 | 高級感を演出するため広い余白を採用する |
