> 作成者ペルソナ：UXデザインディレクター / プロダクトUIアーキテクト / SEOメディアデザインストラテジスト

本ワイヤーフレームはデザイン装飾ではなく **UI構造・コンポーネント配置・情報優先順位** を定義する。

---

## 1. ワイヤーフレーム設計の基本方針

本サイトはSEOメディア / 比較メディア / 意思決定支援メディアとして設計する。
UI構造は以下の順序で構成する。

```
結論 → 比較 → 解説 → CTA
```

## 10. モバイルワイヤーフレーム方針

| 3 | ランキング |
| 4 | CTA |
