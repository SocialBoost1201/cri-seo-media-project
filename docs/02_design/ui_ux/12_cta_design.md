# 転職SEOメディア構築プロジェクト
CTA設計書

> 作成者ペルソナ：Conversion Optimization Strategist / Growth Product Manager / UXコンバージョンアーキテクト

本メディアのCTAは広告バナー型ではなく **「意思決定支援型CTA」** として設計する。

```
検索流入 → 情報提供 → 比較 → 意思決定 → 求人送客
```

---

## 1. CTA設計の基本方針

| 原則 | 説明 |
|------|------|
| 自然導線 | 記事読了の流れを阻害しない |
| 意思決定支援 | 比較情報の直後にCTAを配置する |
| 複数配置 | 記事内に3〜5箇所のCTAを設置する |
| 信頼性 | 広告感を排除する |

---

## 2. CTAタイプ

| CTAタイプ | 用途 |
|----------|------|
| 比較CTA | 比較ページへの誘導 |
| サービスCTA | 転職エージェント紹介 |
| 求人CTA | 求人サービス送客（Webist） |
| 記事CTA | 関連記事誘導 |

---

## 3. CTA配置設計（記事ページ）

| 配置位置 | タイミング |
|---------|----------|
| 記事冒頭 | ユーザーの即時行動に対応 |
| 比較表直下 | 比較検討後の行動誘導 |
| 記事中盤 | 読了途中の離脱防止 |
| 記事末尾 | 読了後の行動誘導 |

---

## 4. CTA文言例

| CTAタイプ | 文言例 |
|----------|-------|
| 比較CTA | おすすめ転職エージェントを見る / IT転職エージェント比較を見る |
| サービスCTA | このエージェントを見る / 詳細を見る |
| 求人CTA | 求人を見る / 転職サービスを見る |
| 汎用 | おすすめ求人を見る / 無料で転職相談する |

---

## 7. CTA心理設計

ユーザー心理に合わせたCTA配置タイミング：

```
情報収集
↓
比較 ← CTA配置ポイント
↓
判断 ← CTA配置ポイント
↓
行動
```

---

## 8. 比較ページCTA（最重要）

比較テーブル内に設置するCTA。

| CTA文言 | 動作 |
|---------|------|
| 公式サイトを見る | 外部リンク（nofollow） |
| 求人を見る | Webistへ送客 |
