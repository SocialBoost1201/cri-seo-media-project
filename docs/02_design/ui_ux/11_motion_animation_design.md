# 転職SEOメディア構築プロジェクト
モーション・アニメーション設計書

> 作成者ペルソナ：Product Motion Designer / UX Motion Director / Web Interaction Architect

本サイトはSEO記事サイトでありながらモダンプロダクトレベルのUI体験を提供する。
過剰な装飾アニメーションではなく **UX改善・情報理解促進・プロダクト品質演出** のためのモーションを採用する。

---

## 1. モーションデザインコンセプト

| 項目 | 内容 |
|------|------|
| コンセプト | **Calm Intelligence Motion** — 落ち着いた知的な動き |
| 印象 | プロダクト品質を感じさせる / 自然で滑らかなアニメーション |
| 参考プロダクト | Apple / Linear / Stripe / Vercel / Notion |

---

## 2. モーション原則

| 原則 | 詳細 |
|------|------|
| 自然な加速 | `ease-out` を基本とする |
| 短時間 | 200ms〜400ms |
| UI理解を助ける | 意味のないアニメーションは禁止 |
| 軽量 | パフォーマンス優先 |

---

## 3. 使用ライブラリ

| 優先度 | 技術 |
|-------|------|
| 推奨 | Framer Motion |
| 代替 | CSS Motion（CSS Transitions / Animations） |

---

## 4. アニメーション定義

### 4.1 ヒーローセクション（TOPページ）

| 要素 | 動き | duration |
|------|------|----------|
| タイトル | フェードイン + 上方向スライド | 600ms |
| サブタイトル | フェードイン + 上方向スライド | 600ms（ディレイ付き） |
| CTA | フェードイン + 上方向スライド | 600ms（ディレイ付き） |

### 4.2 記事カード（ホバー）

| プロパティ | 値 | duration |
|-----------|---|----------|
| scale | 1.03 | 200ms |
| shadow | 強化 | 200ms |

### 4.3 カテゴリカード（ホバー）

| プロパティ | 値 | duration |
|-----------|---|----------|
| 背景色 | 変化 | 200ms |
| アイコン scale | 1.05 | 200ms |

### 4.4 スクロール表示

| 要素 | 動き | duration |
|------|------|----------|
| 記事カード | フェードイン + translateY(20px → 0) | 500ms |

### 4.5 比較テーブル

| 要素 | 動き | duration |
|------|------|----------|
| テーブル全体 | フェードイン | 400ms |
| 各行 | 順次表示 | 50msディレイ |

### 4.6 CTAボタン

| アクション | プロパティ | 値 |
|-----------|-----------|---|
| ホバー | scale | 1.05 |
| ホバー | shadow | 強化 |
| クリック | scale | 0.97 |

### 4.7 FAQアコーディオン

| 動き | duration | easing |
|------|----------|--------|
| 高さアニメーション | 250ms | ease-out |

### 4.8 ヘッダー（スクロール時）

| 動き | duration |
|------|----------|
| 背景フェード（透明 → 白） | 200ms |

### 4.9 ページ遷移

| 動き | duration |
|------|----------|
| opacity 0 → 1 | 200ms |

## 7. アニメーション禁止事項

| 禁止 |
|------|
| 過剰なパララックス |
| 長いアニメーション（>600ms） |
| 派手なエフェクト |
| SEOサイトのUXを阻害する演出 |

---

## 8. パフォーマンス基準

- Core Web Vitals を阻害しない設計とする
- JSサイズ最小化
- アニメーションはGPU処理（`transform` / `opacity` 中心）
