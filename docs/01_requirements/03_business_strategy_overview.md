## 1. 事業背景

クリーク・アンド・リバー社はクリエイティブ業界を中心とした人材サービス企業であり、
既存の求人メディアとして「Webist」を運営している。

**求人検索型サービスの課題：**

- 検索流入が限定的
- 転職意思決定前のユーザーに接触できない
- SEOコンテンツによる流入が弱い

以下のユーザー行動のうち「**検索上流**」を獲得するメディアが必要となる。

```
検索ユーザー
↓
情報収集
↓
比較
↓
転職意思決定
↓
求人
```

---

## 2. 新規SEOメディアの役割

新規メディアは「**転職情報SEOメディア**」として設計する。

| メディア | 位置付け | 目的 |
|---------|---------|------|
| Webist | 求人プラットフォーム | 求人掲載・応募受付 |
| SEOメディア（本PJ） | 検索流入エンジン | 検索流入獲得 / 転職意思決定支援 / 求人サービス送客 |

---

## 3. ユーザー導線設計

```
Google検索
↓
SEO記事
↓
転職情報
↓
転職エージェント比較
↓
求人情報
↓
Webist
```

この導線を構築することで、検索上流ユーザーを効率的に送客できる。

| キーワード | 月間検索ボリューム（推定） |
|-----------|--------------------------|
| 転職 | 20万〜40万 |
| 転職エージェント | 約10万 |
| IT転職 | 約3万 |

| 転職関連（ロングテール含む） | **月間100万以上** |

## 5. 競争環境と戦略

### 主な競合

| 区分 | 媒体名 |
|------|-------|
| 大手求人メディア | リクナビNEXT / マイナビ転職 / doda / ビズリーチ |
| SEOメディア | ミライトーチ / キャリアガーデン / 転職アンテナ |

上記は非常に強力なドメインを持つため、総合転職メディアとして正面から戦うのは困難。
そのため **「特化カテゴリ戦略」** を採用する。

---

## 6. カテゴリ戦略

| カテゴリ | 選定理由 |
|---------|---------|
| 転職エージェント | 検索需要高 / CV単価高 |
| IT転職 | 市場成長 / 需要安定 |
| 第二新卒転職 | ターゲット明確 / コンテンツ化しやすい |
| 営業転職 | 検索需要高 / 転職意思決定に近い |
| ハイクラス転職 | 高単価送客 / 差別化可能 |

---

## 7. 収益モデル

| 収益源 | 詳細 |
|-------|------|
| 転職エージェント送客 | ASP単価 5,000〜30,000円 |
| Webist求人送客 | 自社求人への内部送客 |

SEOメディア → 求人サービスという **内部送客モデル** を構築する。

## 9. SEO資産化戦略

本メディアは広告メディアではなく「**検索資産メディア**」として構築する。

| 特徴 |
|------|
| 検索流入が継続する |
| コンテンツ資産が蓄積する |
| 広告費不要の集客が可能 |

中長期的には **月50万PV以上のメディア** へ拡張可能。
