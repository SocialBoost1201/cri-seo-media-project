# 転職SEOメディア構築プロジェクト
企画提案サマリー

## 1. プロジェクト概要

本プロジェクトは、株式会社クリーク・アンド・リバー社が保有する求人メディア「Webist」とは別に、
SEO流入を獲得する転職情報メディアを構築し、検索ユーザーを求人サービスへ送客することを目的とする。

既存の求人メディアは「求人検索」が中心であり、
検索エンジンからの上流ユーザーの獲得には限界がある。

そのため本プロジェクトでは以下の検索流入導線を設計し、
SEOメディアを「送客エンジン」として構築する。

```
検索
↓
転職情報
↓
転職サービス比較
↓
求人
↓
Webist
```

---

## 2. プロジェクト目的

本メディアの主目的は以下の3つである。

1. 検索エンジンから転職検討ユーザーを獲得する
2. 転職エージェント・転職情報コンテンツを提供し意思決定を支援する
3. 最終的にWebistの求人へ送客する

---

## 3. 既存サービスとの役割分担

| 区分 | メディア名 | 役割 | 主な機能 |
|------|-----------|------|---------|
| 既存 | Webist | 求人検索メディア | 求人検索 / 求人応募 / 転職支援 |
| 新規 | SEOメディア（本プロジェクト） | 検索流入獲得メディア | 転職情報コンテンツ / 転職エージェント比較 / キャリア情報記事 / 転職ノウハウ記事 |

---

## 4. SEOメディアのポジション

本メディアは一般的なアフィリエイトメディアではなく、**「転職意思決定支援メディア」** として設計する。

検索ユーザーの転職検討フェーズにおいて、以下を支援するコンテンツを提供する。

- 情報収集
- 比較
- 意思決定

---

## 5. 想定カテゴリ

本メディアでは以下のカテゴリを中心に構成する。
これらは検索需要が高く、かつWebistの既存事業と競合しない領域として選定する。

- 転職エージェント
- IT転職
- 第二新卒転職
- 営業転職
- ハイクラス転職

---

## 6. SEO戦略の基本方針

1. 検索意図ベースのコンテンツ設計
2. 比較記事を中心としたCV導線
3. 内部リンクによるカテゴリ強化
4. E-E-A-T強化

---

## 7. GEO（AI検索）対策

近年の検索環境ではAI検索への最適化が重要である。
以下の要素を記事構造に組み込むことで、AI検索への引用可能性を高める。

| 要素 | 目的 |
|------|------|
| 比較表 | 構造化情報の提供 |
| ランキング | 選択肢の整理 |
| FAQ | 質問意図への直接回答 |
| 定義コンテンツ | 正確な情報ソースとしての認知 |
| 要約セクション | AI引用の起点 |

対象AI検索：ChatGPT / Perplexity / Gemini
