# 転職SEOメディア デモサイト
要件定義書

> 作成者ペルソナ：プロダクトマネージャー / SEOメディア戦略ディレクター / UXアーキテクト / GEO最適化コンサルタント

---

## 1. プロジェクト目的

1. SEOメディア構造の実装イメージを提示する
2. 検索流入から求人メディア送客までの導線を可視化する
3. クリーク・アンド・リバー社の既存サービス「Webist」との役割分担を明確にする
4. SEO・GEO最適化された転職メディアの構造を提示する

---

## 2. プロジェクトのゴール

- SEOメディア構造の理解
- 転職ユーザー導線の可視化
- UI/UX体験の提示
- 事業拡張可能な設計の提示
